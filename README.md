# hubot-jazzbot-http

A hubot script.

## Installation

In hubot project repo, run:

`npm install hubot-jazzbot-http --save`

Then add **hubot-jazzbot-http** to your `external-scripts.json`:

```json
[
  "hubot-jazzbot-http"
]
```
